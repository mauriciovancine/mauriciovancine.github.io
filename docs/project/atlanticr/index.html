atlanticr

Pacote R para acessar dados do ATLANTIC Series, um conjunto de data papers sobre biodiversidade e dados geoespaciais da Mata Atlântica.

Autor

Maurício Vancine, Bernardo Niebuhr, Renata L. Muylaert, Mauro Galetti, Milton Cezar Ribeiro

Website

atlanticr

As florestas tropicais e subtropicais ao longo da costa e do interior da América do Sul costumavam ter uma das maiores florestas do mundo – a Mata Atlântica – com mais de 1,6 milhão de km² abrangendo Brasil, Argentina e Paraguai. O ATLANTIC: Data Papers from a biodiversity hotspot inclui uma série de Data Papers sobre a composição da comunidade, distribuição de plantas e animais e características ambientais da Mata Atlântica. Esses artigos são uma ferramenta fundamental para cientistas e formuladores de políticas.

O objetivo deste pacote é fornecer dados do ATLANTIC, uma série de artigos de dados da Ecology sobre dados de biodiversidade da Mata Atlântica.

Atualmente, existem 17 conjuntos de dados: nove sobre vertebrados, dois sobre características de vertebrados, dois sobre invertebrados, três sobre interações animal-planta, um sobre planta e um sobre dados espaciais.

Instalação

Você pode instalar a versão de desenvolvimento do atlanticr do GitHub com:

# install.packages("remotes")
remotes::install_github("mauriciovancine/atlanticr")

Conjuntos de dados

Os conjuntos de dados atualmente incluídos no pacote estão resumidos abaixo; consulte os artigos individuais para obter detalhes sobre dados e fontes.

Vertebrados

  • atlantic_amphibians: ATLANTIC AMPHIBIANS: a data set of amphibian communities from the Atlantic Forests of South America

  • atlantic_batsATLANTIC BATS: a data set of bat communities from the Atlantic Forests of South America

  • atlantic_birds: ATLANTIC BIRDS: a data set of bird species from the BrazilianAtlantic Forest

  • atlantic_camtraps: ATLANTIC-CAMTRAPS: a dataset of medium and large terrestrial mammal communities in the Atlantic Forest of South America

  • atlantic_camera_trap_mammals: Camera trap surveys of Atlantic Forest mammals: A data set for analyses considering imperfect detection (2004–2020)

  • atlantic_mammals: ATLANTIC MAMMALS: a data set of assemblages of medium- and large-sized mammals of the Atlantic Forest of South America

  • atlantic_nonvolant_mammals: Non-volant mammals from the Upper Paraná River Basin: a data set from a critical region for conservation in Brazil

  • atlantic_primates: ATLANTIC-PRIMATES: a dataset of communities and occurrences of primates in the Atlantic Forests of South America

  • atlantic_small_mammal: Atlantic small-mammal: a dataset of communities of rodents and marsupials of the Atlantic forests of South America

  • atlantic_small_mammal_abundance: Abundance of small mammals in the Atlantic Forest (ASMAF): a data set for analyzing tropical community patterns

Traits de vertebrados

  • atlantic_amphibian_traits: ATLANTIC AMPHIBIAN TRAITS: a data set of morphological and ecological traits of amphibians in the Atlantic Forest

  • atlantic_bird_traits: ATLANTIC BIRD TRAITS: a data set of bird morphological traits from the Atlantic forests of South America

  • atlantic_mammal_traits: ATLANTIC MAMMAL TRAITS: a data set of morphological traits of mammals in the Atlantic Forest of South America

Invertebrados

  • atlantic_ants: ATLANTIC ANTS: a data set of ant communities from the Atlantic Forests of South America

  • atlantic_butterflies: Atlantic butterflies: a data set of fruit-feeding butterfly communities from the Atlantic forests

Interação animal-plants

  • atlantic_frugivory: Atlantic frugivory: a plant–frugivore interaction data set for the Atlantic Forest

  • atlantic_pollination: ATLANTIC POLLINATION: a data set of flowers and interaction with nectar-feeding vertebrates from the Atlantic Forest

  • atlantic_flower_invertebrate_interactions: Atlantic flower–invertebrate interactions: A data set of occurrence and frequency of floral visits

Plantas

  • atlantic_epiphytes: ATLANTIC EPIPHYTES: a data set of vascular and non-vascular epiphyte plants and lichens from the Atlantic Forest

Espacial

  • atlantic_spatial: ATLANTIC SPATIAL: a data set of landscape, topographic, hydrological, and anthropogenic metrics for the Atlantic Forest
