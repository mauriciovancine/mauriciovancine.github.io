GeoR: Dados Geoespaciais com o R

Disciplina de pós-graduação de manipulação e visualização de dados geoespaciais

website code

Descrição

Na GeoR, os alunes serão apresentados aos principais conceitos teóricos e práticos do funcionamento da linguagem R para o uso para manipulação e visualização de dados geoespaciais. Serão abordados temas como controle de versão, git e GitHub; estrutura e fonte de dados geoespaciais; e visualização de dados geoespaciais.

Oferecimento

  2. Programa de Pós-graduação em Ecologia, Evolução e Biodiversidade. (2020). Universidade Estadual Paulista (Unesp), Rio Claro, SP, Brasil.

  3. Programa de Pós-graduação em Etnobiologia e Conservação da Biodiversidade. (2019). Universidade Federal Rural de Pernambuco (UFRPE), Recife, PE, Brasil.
